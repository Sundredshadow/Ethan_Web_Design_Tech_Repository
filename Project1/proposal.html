I.High Concept

A small, intense, story driven, game focused on cyber security and hacking. Using actual hacking principals, alongside fun game modes and content, is the primary game mechanic

II.Genre

Puzzle
Educational
Story Driven

III.Platform

Desktop (to start with maybe mobile)

IV.Story

Will focus on actual hacking scenarios that are applicable to the average person. Will have the player act as the hacker, and have them break through a firewall.

V.Esthetics

Graphics: Pixel/2D; 8bit-32bit top-down shooter. Mimic's a firewall

VI.Gameplay

Since this is a game that integrates cyber security principals, with regular gameplay, it will descriptions on what is happening alongside the traditional gameplay. These description may include information on enemies that will influence later actions within the game.
